对象思维

类

public,公有，全都可以访问。
private，自己和友元可以访问。
protect，自己，友元，派生可以访问。

private和protect的不同在于：派生.其实protect就是为了能够继承，而且又能保证子类不对外开放，即该函数不能作为外部接口，只能作为自己使用。

在继承的时候各个意思

(1)所有的private只能是自己可以访问，不管你是谁
(2)public继承说明公有继承，就是除了private的东西其他子类可以访问，而且对于子类来说public对应的就是public，protect对应的就是protect。
(3)protect继承，和public不同之处在于，子类继承下来以后对于他是protect的。也就是外部不能访问子类了
(4)private继承，和public，protect不同在于，子类继承下来以后对于他是private的。只有他自己可以访问，其他的人一律不行。

this指针一般用于当前对象自身。

什么是子对象，就是该成员是另一个对象的成员。

实验

#include <stdio.h>
#include <stdlib.h>

class ClassA {
private:
int a;
int b;

public:
void Public_Funtion(void)
{
printf("%s\n", "Funtion in A of");
}

protected:
void Protected_Funtion(void)
{
printf("%s\n", "Funtion in protected");

}
};

class ClassB : public /* protect */ClassA{
private:

public:
void Public_Funtion_B(void)
{
printf("%s\n", "Funtion in punilc of B");
Protected_Funtion();
}

};
void main()
{
ClassB A;
A.Public_Funtion_B();
}

protect和public继承都行。
