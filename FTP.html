    • 什么是uri

    Uniform Resource Identifier 统一资源标识符
    网络上的每种资源都通过url进行定位，由三部分组成，通过这三部分即可获取资源路径。

    URL是URI命名机制的一个子集

    一，主机名
    存放资源自身的名称，由路径表示。
    协议名称://域名.根域名/目录/文件名.后缀
    如:http://baike.so.com/doc/100.html 这是可通过HTTP协议访问的资源，位于baike.so.com的主机上。在/doc文件夹，100.html的这个文件。

    二，标识符
    协议://域名/目录/文件#片段标示符（例如：/a/b.php#a）

    三、相对URI

    安装ftp

    yum install vsftpd*

    启动

    /sbin/service vsftpd start

    禁止匿名登录

    vim /etc/vsftpd/vsftpd.conf
    anonymous_enable=NO

    设置开机启动

    chkconfig --level 35 vsftpd on

    创建用户名

    useradd zhPeServer

    ftp 127.0.0.1 登录本机ftp

    访问ftp的哪个目录

    ftp进去的目录需要查看登录的账号的目录的主目录。
