作用：scanf（a，b）；键盘上的数据转换成a的格式放到b里面。

1，*放在格式里表示只读改格式的值，但不会赋值给变量。

2，%d，%f等只要不是字符格式的，只要遇到不是字符或者空格就认为读的任务结束。-

#include <stdio.h>

int main(void)
{
int x, y;
scanf("%d%d", &x, &y);
printf("%d\n%d", x, y);
return 0;
}

解释：当格式为%d的时候如果遇到一个不是数字或者空格（包括tab键等）就解释成已经结束输入。

#include <stdio.h>

int main(void)
{
int x;
char y;
scanf("%d%c", &x, &y);
printf("%d\n%c", x, y);
return 0;
}

%c会接受空格。所以要在%d%c之间插一个以上的空格（包括tab那些），它就回跳过你按的entered键。

#include <stdio.h>

int main(void)
{
char x, y;
scanf("%c%c", &x, &y);
printf("%c\n%c", x, y);
return 0;
}
linwei@linwei-virtual-machine:~/lesson$ ./c
1
1

linwei@linwei-virtual-machine:~/lesson$

从这可看出scanf先把1当作第一个字符，读进去，然后如果按enter则把enter读进去，后面根本就读不到了

scanf返回值

scanf返回的是成功转换的并存储于参数中的值的个数。
