作用：scanf（a，b）；键盘上的数据转换成a的格式放到b里面。

1，*放在格式里表示只读改格式的值，但不会赋值给变量。

2，%d，%f等只要不是字符格式的，只要遇到不是字符或者空格就认为读的任务结束。-

#include <stdio.h>

int main(void) { int x, y; scanf("%d%d", &x, &y); printf("%d\n%d", x, y); return 0; } linwei@linwei-virtual-machine:~/lesson$ ./c a b 134513801

#include <stdio.h>
