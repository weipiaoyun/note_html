rhel5.5配置yum源

环境，没有任何xml文档。如果有xml文档用上面那种方法
1，拷贝Server的所有rpm包到/opt目录下面
2，修改/etc/yum.repos.d/mylocal.repo为如下：
[local_resource]
name=my_local_resource
baseurl=file:///opt/Server enable=1
gpgcheck=0
3,vim /usr/lib/python2.6/site-packages/yum/yumRepo.py``
将remote = url + '/' + relative改为remote = "/media/rhel_local_resources" + '/' + relative

4,yum clean all
yum grouplist
yum list
