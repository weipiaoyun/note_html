API

Web API：用来写HTTP API的开源框架。是可以通过使用HTTP协议使用的通过web实现的API。它只是一个概念，不是一个技术。它可以通过各种语言，C#，java或其他来实现。Web API可以是RESTful或其他。 web API实现了协议的规格。所以它包括了caching,URLs，versioning,request/response headers,和很多的内容格式

SOAP API：Simple Access Protocol

Web Service：facilitates interaction between two machines over a network
API:an interface between two different applications so that they can communicate with each other.

1. All Web services are APIs but all APIs are not Web services.
2. Web services might not perform all the operations that an API would perform.
3. A Web service uses only three styles of use: SOAP, REST and XML-RPC for
communication whereas API may use any style for communication.
4. A Web service always needs a network for its operation whereas an API doesn’t need a network for its operation.
