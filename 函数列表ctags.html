如何安装

1，下载配置文件
taglist_46.zip，将配置拷贝的相应的文件夹

2，下载ctags.exe执行文件

3，把ctags.exe设置环境变量

4，在vimrc里面设置ctags路径及配置，注意，位置不要调换
set tags=tags;
set tags+=D:/tags
let Tlist_Ctags_Cmd='D:\vim\vim73\ctags.exe'
let Tlist_Auto_Open = 1
set autochdir

5，在D:下面建立tags文件夹

命令

u 刷新函数
s 更改排序方式
x taglist窗口放大和缩小，方便查看较长的tag
+ 打开一个折叠，同zo

  • 将tag折叠起来，同zc
  • 打开所有的折叠，同zR

= 将所有tag折叠起来，同zM
[[ 跳到前一个文件
]] 跳到后一个文件
q 关闭taglist窗口

注意

ctags执行后会在tags目录下面形成一个tags文件
要把模块加进去才能把函数写入tags里面，找的时候才能找得到

ctrl+鼠标左键/鼠标右键
