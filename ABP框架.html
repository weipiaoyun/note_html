DDD

DDD is a set of Guiding Principles
The primary principles are focused on the core domain explore models in a creative collaboration.

software experts and domain experts.

There are four fundamental layers in Domain Driven Design

  • Presentation Layer ：like windows，it provides an interface to the user.
  • Application Layer : Medias between the Presentation and Domain Layers.
  • Domain Layer : Includes business objects and their rules. This is the heart of the application.
  • Infrastructure Layer : Provides generic technical capabilities that support higher layers mostly using 3rd-party libraries.

Presentation Layer

ASP.NET [Core] MVC (Model-View-Controller) can be considered to be the presentation layer
It include

  • Localization
  • Navication
  • Object Mapping
  • Caching
  • Configuration Management
  • oAudit Logging
