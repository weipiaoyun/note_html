GME-WF1-001的采集

设备与MES对接要求
1.       厂商设备须有条码扫描功能，能扫描读取工单号，并与建立与MES对接的接口中读取对应工单工件的所有参数用于生产中（有效防止操作人员输入错误）
2.       设备实时的运行情况记录到MES中（入料,出料,故障报警,温度,电流等）
3.       MES提供相应的接口，与厂商设备进行对接。
提供给厂商接口技术（注： 基于网络安全考虑一般采用webservice ），提供内网与外网的测试地址 ，各个接口的名称与参数的个数,内容定义，数据格式，长度。（通常需要与设备商双方沟通先确定下来，因为每个流程的设备需求不一样！）
4.    DMS, MES均为以太网接口, 配8口交换机
 
 
注：提供内外网的测试地址非常重要可以更快更有效开展工作（与深南电路也是这样做的，主导是你们MES开发人员与工艺人员，每个板件资料必须要在MES系统中维护进来的，同时还提供了不同开发语言的对接案例给厂商参考）。

  --1查找厂别， 2，查看当前流程。3，查找MI流程。4
  --SELECT CORG_CODE FROM dbo.TBL_PLAN_JOB_TICKET WHERE CWORK_ORDER_NUMBER = 'B0024610-1-87-42'

  DECLARE @CWORK_ORDER_NUMBER_RANGE VARCHAR(MAX) = '46-61'
  DECLARE @CWORK_ORDER_NUMBER VARCHAR(MAX) = 'B0024338-0-32-22A'
  DECLARE @PROCESS_CODE  VARCHAR(MAX) = 'SLDR2'
  DECLARE @CSTEP  INT = 7


  DECLARE @PROCESS_CID VARCHAR(MAX)
  DECLARE @CWORK_ORDER_NUMBER_CID VARCHAR(MAX)
  
  DECLARE @LEFT  INT
  DECLARE @RIGHT  INT

 SET @LEFT =  CONVERT(INT,LEFT(@CWORK_ORDER_NUMBER_RANGE,2))
 SET @RIGHT = CONVERT(INT,RIGHT(@CWORK_ORDER_NUMBER_RANGE,2))

 WHILE (@LEFT <= @RIGHT)
 BEGIN
  DECLARE @CWORK_ORDER_NUMBER_NEW VARCHAR(MAX)
   SET  @CWORK_ORDER_NUMBER_NEW = @CWORK_ORDER_NUMBER + CONVERT(VARCHAR(MAX),@LEFT)
   SELECT @PROCESS_CID = CID FROM dbo.TBL_BD_PROCESS WHERE CPROCESS_CODE = @PROCESS_CODE
    SELECT @CWORK_ORDER_NUMBER_CID = CID FROM dbo.TBL_PLAN_JOB_TICKET WHERE CWORK_ORDER_NUMBER = @CWORK_ORDER_NUMBER_NEW
	SET @LEFT = @LEFT + 1
   UPDATE dbo.TBL_PLAN_JOB_TICKET_ONLINE SET CSTEP = @CSTEP, CDEPT_PTR = @PROCESS_CID  WHERE CWO_PTR = @CWORK_ORDER_NUMBER_CID
 END



 --单独设置
DECLARE @PROCESS_CODE  VARCHAR(MAX) = 'FQA1'
DECLARE @CWORK_ORDER_NUMBER_NEW VARCHAR(MAX) = 'B0024338-0-32-22B'
DECLARE @PROCESS_CID VARCHAR(MAX)
DECLARE @CWORK_ORDER_NUMBER_CID VARCHAR(MAX)
SELECT @PROCESS_CID = CID FROM dbo.TBL_BD_PROCESS WHERE CPROCESS_CODE = @PROCESS_CODE
SELECT @CWORK_ORDER_NUMBER_CID = CID FROM dbo.TBL_PLAN_JOB_TICKET WHERE CWORK_ORDER_NUMBER = @CWORK_ORDER_NUMBER_NEW
UPDATE dbo.TBL_PLAN_JOB_TICKET_ONLINE SET CSTEP = 0, CDEPT_PTR = @PROCESS_CID  WHERE CWO_PTR = @CWORK_ORDER_NUMBER_CID
