利用putty实现免用户和免密码登陆

1，运行putty key generator 点击Generate(使用默认的SSH-2 RSA) 移动鼠标完成进度 生成public和private key 复制出生成的key pasting into Open SSH authorized_keys file 点击保存私匙(公匙可以保存，可以不保存)

2，运行putty configuration 在Auth里面的 private key file for authentication 浏览选择生成的私匙 在connection 里面点击data 在auto-login usename里面填root
