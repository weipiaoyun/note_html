geometry manager controls the position and the size. frame or toplevel widget(MainWindow is special toplevel) widget separate groups of widgets(they are master);

it 's difficult to use more than one geometry manager within the same area.(so if we start using pack, we should continue to use pack, because two geometry managers will get into a race condition and sometimes it will goto die while).if we want to use pack and grid at the same time,we can use the Frame.

The geometry manager have pack, grid ,form and place.

pack

pack is the most commonly used, pack allows you to control: 1, position in the window relative to the window or Frame edges 2, Size of widgets, relative to other widgets or absolute 3, spacing between widgets 4, position in the window's or Frame's widget list

widgets cannot overlap or cover each other,partially and completely and it only with rectangular shapes!

for the pack, order is important

pack -widget->packForget();if want to pack again，you can use "-widget->pack()"; -side => 'top' #the defaule size is top -after => $widget1 #在widget1的后面 -ipadx => 10m #internal 本身的widget长度10mm -ipady => 20m #internal 本身的widget宽度20mm -padx => 10 #spacing of the widget -pady => 10 #spacing of the widget -expand => '1' -anchor => 'w' #the default of anchor is center; when use anchor, the defaule of expand is 1 -in, displaying in a parent other than your own. sometimes when -side and -anchor use together, the result won't what we wan't to see

-expand和-fill 1，没有expand和没有fill，则top放在上面，缺省大小为所需大小。 2，有fill，没有expand，它会填充，但是，top和bottom只能左右填充(保证上下位置不变，top还是top，bottom还是bottom)，left和right只能上下填充(保证左右位置不变，left还是left，right还是right)。 3, 有expand没有fill。占用的位置会扩展，但本身的控件不扩展。 4，有expand和有fill。要看顺序，如果第一个填充。第二个如果是同类型的，比如top和bottom。left和right。则可以平分。否则第二个只满足足够大小。
