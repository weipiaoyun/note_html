配置/var/named/chroot/var/named/zh.db

安装配置DNS

安装

yum install bind yum install bind-chroot

修改DNS主配置文件

vi /etc/named.conf 和/var/named/chroot/etc/named.conf 是相同的文件

修改 listen-on port 53 {127.0.0.1;}; 为 listen-on port 53 {any;};

注释掉 //listen-on-v6 port 53 {::1;};

修改 /etc/named.rfc1912.zones文件

增加 zone "zh." IN { allow-update {192.168.0.0/16;}; type master; file "zh.db"; }

cp /var/named/chroot/var/named.localhost /var/named/chroot/var/zh.db

注意更改权限

chmod 777 /var/named/chroot/var/zh.db

修改 /var/named/chroot/var/zh.db

修改 rname.invalid.为ns.cbf.com.

后面增加 cam10 IN A 192.168.6.10 cam06 IN A 192.168.6.6 cam04 IN A 192.168.6.4

什么是正向解析文件和反向解析文件

禁用防火墙 重启 /sbin/service named restart
