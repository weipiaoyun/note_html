Chomp命令，如果是标量，如果最后一行是空行，其会咬掉，如果非空行，不作任何操作。如果是数组，它会检测每个标量的最后一行，如果是空行，就咬掉。返回值：如果咬掉一个标量的空行，就返回1，如果咬掉10个标量的空行，就返回10，没咬，就返回0.只能咬掉最后一个\n

foreach grep map

这三个都会逐个取值(取数组里面的值)。 map和grep不同的地方在于map是处理后的值赋值给新的数组，而grep只是过滤数组。(一个是处理，一个是过滤)，grep里面有个判断。

grep过滤，map获取新数组

grep现有列表，挑选出新列表。循环给值，符合赋值。组成数组。(参数为一个判断(通常正则)，一个数组)。标量上下文，个数。列表上下文，数组。 map, 现有列表延展出新列表。map的意义在于获得新列表，最好不要在里面修改值，也是两个参数，一个是处理表达式，一个是数组。 map,值为列表或列表个数, 循环给值，变为列表 map，转换数组到hash map ({list1} list2); #值是list1的列表或list1的列表个数。list1如果用大括号括住就不能使用逗号，当list1只有一个元素的时候，可以用大括号，也可以使用逗号。多个的时候只能是大括号。list2可以是@a1, @a2, @a3等多个数组组成的列表 @selected = grep EXPR, @input; @selected = map { if (EXPR) { $_ } } @input; my @files1 = map { /(.*)\.txt$/ } @files;#如果匹配，则返回的时圆括号里面的内容。

需要一个列表推演到另外一个列表，用map 筛选列表元素时，用grep函数。 遍历需要修改变量，用foreach

常用 1，求出数组中不一样的 my @bb; my %diff; my @diff = grep (!\(diff{\)_}++, @bb);

2,求数组中两个以上的 my @array; my %same; my @same = grep (\(same{\)_}++, @array);

3，求某个值是不是数组的元素 if (exists \(h->{\)type}{$layerName})

两数组交集，并集，补集

#!/usr/bin/perl -w use Genesis; my $f = new Genesis; my @a=("a","b","c","d","e"); my @b=("b","g","f","e");

my %a=map{$_=>1} @a; my %b=map{$_=>1} @b;

  1. 交集

@A = grep( \(a{\)_}, @b ) ;

  1. 补集

@B1=grep(!exists \(a{\)_}, @b); @B2=grep(!exists \(b{\)_}, @a);

  1. 并集

%C=map {$_=>1} @a,@b; @C=keys %C;

$f->PAUSE("a:@A,b1:@B1, b2:@B2, c:@C");

$a{abcd} = 1;
