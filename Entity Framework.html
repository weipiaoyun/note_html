Entity Framework is an object-relational mapper (O/RM) that enables .NET developers to work with a database using .NET objects。

Entity Framework Core

Getting Started with EF Core

  1. 新建一个.net Core控制台应用程序。名字为：EFGetStarted
  2. Install EF Core sqlserver支持.在Tools > NuGet Package Manager > Package Manager Console包管理终端运行 Install-Package Microsoft.EntityFrameworkCore.SqlServer -Version 3.1.1
  3. 在PMC(Package Manager Console)里面添加EFCore工具，执行命令 Install-Package Microsoft.EntityFrameworkCore.Tools 使得如下的Add-Migration和Update-Database命令在PMC里面可以找到命令。
  4. 添加Migration，执行命令 Add-Migration InitialCreate 。其中的InitialCreate名字可以更换。
  5. 然后执行数据库操作，执行命令 Update-Database
  6. 如果要更新表，更改完model后(可以删除和添加)，重新运行第4，第5步骤即可。第4步的名字需要更换。

Entity Framework Core tools reference

  • Install-Package Microsoft.EntityFrameworkCore.Tools
  • Update-Package Microsoft.EntityFrameworkCore.Tools
  • Get-Help about_EntityFrameworkCore

问题处理

Microsoft.EntityFrameworkCore.Tools版本不对

  1. Install the latest .NET Core SDK.
  2. Update Visual Studio to the latest version. Update-Package Microsoft.EntityFrameworkCore.Tools
  3. Edit the .csproj file so that it includes a package reference to the latest tools package, as shown earlier.
