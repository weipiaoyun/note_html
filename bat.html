连接网络盘

net use \\172.17.1.98 iasystem /user:g2k net use \\172.17.3.72 iasystem /user:g2k

cmdkey /add: 172.17.1.98 /user:%computername%\g2k /pass:iasystem cmdkey /add: 172.17.3.72 /user:%computername%\g2k /pass:iasystem pause install ok
