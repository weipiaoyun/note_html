\(genesis->PAUSE("@{\)$cc{bb}}");

%aa = (bb => [a, c, b], ee => ff);一键对一值，bb => [a, c, b]中的[a, c, b]是个指向数组的引用，类型为scale。注意：里面的时一一对应关系，不会是一对多。
名称表地址$cc表示的是aa
键值有两种情况：标量数值，数组或哈希的引用。

genesis是一个scalar，其是一个变量名，内容为0x1d9b86c，是个引用,指向一个无名哈希。$\(genesis{doinfo}或者\)genesis->{doinfo}，为哈希值，其也是一个scalar，就是一个地址，这地址指向无名哈希，通过该地址可求出某个键的值。

%aa = (bb => [hong, tian, liang], ee => ff);
my %tmp;
@tmp{@{$aa{"bb"}}} = qw(ha hi ho);
@known_opt{@KnownOptIn, @KnownOptOut} = ("aaa") x 100;

print Dumper(@tmp{hong, tian});##打印出来时ha，hi
如果是取哈希的单个值，直接用$tmp{}表示，如果是取哈希的多个值，那么用@tmp{多个键值}，表示的是多个键所对应的值所组成的数组
变量前面加反斜杠，表示取址

sort 子程序或代码块list，后面必须要用括号扩住。

\(h->{aa}->{bb}.\)h是引用第一个箭头不能省略，如果省略编程\(h{aa}->{bb}.\)h不是引用，而是一个变量。由于\(h->{aa}不管怎样只要后面也是下标都表示引用(不像\)h，h有可能是变量，有可能是引用)。故可以省略左右都是下标的箭头。

标量和引用之间很明显的不同在于引用表示某种类型。可以用ref来返回该引用所指向的数据的类型。
